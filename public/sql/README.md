# SQLファイル管理

このディレクトリには、カテゴリ別のSQL文を保存します。

## ファイル命名規則

- ファイル名はカテゴリ名と完全一致させる必要があります
- 拡張子は `.sql` を使用してください

例：
- カテゴリ名が「SLEEP寝具」の場合 → `SLEEP寝具.sql`
- カテゴリ名が「アパレル」の場合 → `アパレル.sql`
- カテゴリ名が「家電」の場合 → `家電.sql`

## 使用方法

1. セッション画面で「SQL」ボタンをクリック
2. カテゴリ名と一致するSQLファイルが自動的に読み込まれます
3. SQLファイルが存在しない場合は、エラーメッセージが表示されます

## サンプルファイル

このディレクトリには以下のサンプルファイルが含まれています：
- `SLEEP寝具.sql` - SLEEP寝具カテゴリ用のSQL
- `アパレル.sql` - アパレルカテゴリ用のSQL
- `家電.sql` - 家電カテゴリ用のSQL

新しいカテゴリを追加する場合は、同じ命名規則でSQLファイルを作成してください。
